## 📌 About

**Lumora** is an AI-powered interactive learning assistant built with Python and Streamlit.  
It is designed to help students, developers, and curious minds learn **any topic** — from basic science to advanced technology — in a personalized and engaging way.

Lumora adapts its explanations, examples, and quiz difficulty based on your chosen skill level: **Beginner**, **Intermediate**, or **Advanced**. Instead of static textbooks, Lumora gives you a dynamic, conversational learning experience powered by Google's **Gemini AI**.

Whether you are preparing for an exam, exploring a new subject, or brushing up on a concept, Lumora is your smart study companion.

---

## ✨ Features

| Feature | Description |
|---|---|
| 🎯 **Adaptive Difficulty** | Choose Beginner, Intermediate, or Advanced — content adjusts automatically |
| 📖 **Smart Explanations** | Clear, structured concept breakdowns tailored to your level |
| 💡 **Real-Life Examples** | Step-by-step analogies from everyday life to make concepts stick |
| ✍️ **Interactive Quiz** | 5 auto-generated questions per topic — MCQ, True/False, and Short Answer |
| 🤖 **AI Answer Grading** | Short answers are evaluated by Gemini — handles synonyms and flexible phrasing |
| 📊 **Instant Feedback** | See your score, correct answers, explanations, and personalised encouragement |
| 🏆 **Completion Badge** | Earn a custom badge every time you finish a quiz |
| 📈 **Score History** | All your past quiz scores are tracked in the sidebar |
| 📋 **Tutoring Summary** | AI generates a full session recap with next learning steps |
| 🔁 **Session Reset** | Start fresh anytime with one click |

## 🚀 Advantages

- **No Prior Knowledge Needed** — Lumora explains any topic from scratch at the Beginner level, making it accessible to everyone.
- **Fully Personalized** — The depth, vocabulary, and complexity of every response adapts to your selected difficulty level.
- **Active Learning** — Instead of just reading, you test yourself with quizzes and receive instant, detailed AI feedback.
- **AI-Powered Grading** — Unlike rigid quiz apps, Lumora understands the intent behind short answers, not just exact keyword matches.
- **Fast & Lightweight** — Built with Streamlit, Lumora runs instantly in any browser with a simple Python command — no complex setup.
- **Secure by Design** — API keys are loaded via `.env` files and never hardcoded or exposed.
- **Self-Correcting** — Smart model fallback logic ensures the app keeps working even if a specific Gemini model version is unavailable.
- **Covers Any Topic** — From Photosynthesis to Neural Networks to Binary Search — if you can name it, Lumora can teach it.
